# Yuruicamp 露營選物品牌網站 - 配色方案

## 核心主色

| 色系 | 顏色代碼 | 應用場景 |
|------|---------|---------|
| **深青綠色** | `#244d4d` | Header、Logo、按鈕、邊框、主文字 |
| **純白色** | `#ffffff` | 背景、按鈕文字、卡片背景 |
| **淺青灰綠** | `#779999` | 連結、副標題、Hover 效果、庫存標籤 |
| **淺綠背景** | `#f6fbf6` | 買家端頁面底色、Header 背景 |

## 強調色

| 色系 | 顏色代碼 | 用途 |
|------|---------|------|
| **深青綠懸停色** | `#316868` | 按鈕 Hover 效果 |
| **深青綠加深** | `#1a3838` | 特殊加深強調場景 |
| **淺青綠選中背景** | `#e8f4f4` | 表單選中項目、日期選取範圍 |
| **金色** | `#d4a017` | 星級評分圖示、推薦標籤 |
| **橘色** | `#e07b39` | 警示訊息、折扣標籤、購物車計數徽章 |
| **淺灰色** | `#f2f2f2` | 商品卡片背景 |
| **大地米色** | `#F2F0EB` | Booking 系統 Header / Offcanvas 背景（與買家端做視覺區分） |

## SCSS 變數定義（css/variables.scss）

```scss
// 主色調
$primary:        #244d4d;   // 深青綠色 - 主色
$primary-dark:   #1a3838;   // 深青綠色加深
$primary-light:  #4a7777;   // 深青綠色淺化

// 副色調
$secondary:      #779999;   // 淺青灰綠 - 副色（連結、標籤）
$secondary-light: #a8bcbc;  // 副色淺化

// 背景色
$white:      #ffffff;
$light-bg:   #f6fbf6;       // 淺綠背景（買家端）
$light-gray: #f2f2f2;       // 淺灰（卡片背景）

// 互動色
$dark-hover: #316868;       // 按鈕 Hover

// 陰影
$box-shadow: 0 4px 6px -1px rgba(0, 0, 0, 0.1);
$dark-shadow: 0 0 0 100vw rgba(0, 0, 0, 0.5);
```

## CSS 變數定義（booking/css/booking.css）

```css
:root {
  --bk-primary:        #244d4d;   /* 深青綠：主色、按鈕 */
  --bk-primary-hover:  #316868;   /* 深青綠懸停色 */
  --bk-primary-light:  #e8f4f4;   /* 極淺青綠：選中背景 */
  --bk-accent:         #779999;   /* 淺青灰綠：副標題、標籤 */
  --bk-bg:             #f6fbf6;   /* 頁面底色 */
  --bk-card-bg:        #f2f2f2;   /* 卡片背景 */
  --bk-gold:           #d4a017;   /* 金色：推薦標籤 */
  --bk-orange:         #e07b39;   /* 橘色：警示、折扣 */
  --bk-danger:         #c0392b;   /* 紅色：錯誤狀態 */
}
```

## 品牌色彩理念

- **主題色 `#244d4d`**：深青綠色傳達自然、沉穩、戶外的品牌形象
- **文字層次**：主文字用深青綠，次要文字用淺青灰綠 `#779999`
- **互動反饋**：Hover 時深化（`#316868`）或轉換色彩
- **卡片設計**：白底 + 淺灰背景 `#f2f2f2` 突顯商品
- **系統區分**：買家端用 `#f6fbf6` 淺綠背景，Booking 系統 Header 用 `#F2F0EB` 大地米色做視覺區隔

適合露營戶外電商的專業、可信賴視覺風格。
